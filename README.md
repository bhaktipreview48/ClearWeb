Once you install ClearWeb, it runs in the background. You can manage your preferences through the browser extensions menu.

*   Click the puzzle piece icon in the top right corner of your browser.
*   Find ClearWeb in the list of installed extensions.
*   Select the settings gear icon to open the configuration panel.
*   Toggle the master switch to enable or disable the filtering engine.
*   Choose specific content categories to block or allow.

## 🚀 Troubleshooting
If you experience issues, consult these common solutions before you seek help elsewhere.

**The browser displays a warning during installation**
Windows SmartScreen might mark new software as unknown. Click "More Info" and select "Run Anyway" to complete the installation.

**Pages do not load correctly**
Some websites rely on scripts that ClearWeb might block. Open the extension menu and use the "Pause on this site" toggle to fix display issues. Refresh the page to reload the content.

**The extension fails to start**
Ensure you have the latest version of your browser installed. An outdated browser can prevent extensions from loading correctly. Check your browser settings to confirm that updates are current.

**ClearWeb consumes too much memory**
If you have dozens of tabs open, your browser may report high resource usage. Close unused tabs to free up system memory and ensure the extension continues to function smoothly.
